# Appendices

The appendices provide reference material, detailed derivations, and practical guides that complement the main text.

They are intended as **lookup resources** rather than sequential reading.

---

## Appendix Overview

- **Appendix A: Mathematical Proofs**  
  Extended proofs and derivations that underpin key results from Part I.

- **Appendix B: Complete API Reference**  
  An overview of core types and functions in OctaIndex3D.

- **Appendix C: Performance Benchmarks**  
  Methodology and selected results for performance evaluation.

- **Appendix D: Installation and Setup**  
  Practical instructions for building and configuring OctaIndex3D.

- **Appendix E: Example Code**
  Self-contained examples illustrating common tasks and integration patterns.

- **Appendix F: Migration Guide**
  Practical guidance for migrating from cubic grids, octrees, or other spatial indexing schemes (H3, S2) to OctaIndex3D.

- **Appendix G: Performance Tuning Cookbook**
  Quick-reference decision guide with "if you see X, try Y" recipes, decision trees, and actionable checklists.

- **Appendix H: Integration Examples**
  End-to-end integration examples with popular tools (nalgebra, Bevy, Python bindings) and frameworks.

- **Appendix I: BCC Quick Reference Card**
  Single-sheet reference with parity formula, neighbor offsets, Morton code, key metrics, and emergency fixes.

- **Appendix J: Troubleshooting Guide**
  Common problems, symptoms, root causes, and solutions. Start here when stuck.
